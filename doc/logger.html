log_enter/1

log_enter(_Tag) -> ok

Logs a message and signals entering a function, block, ...

This function will first print the log message (using the current indentation level) and then increase the indentation level of logs. Don't forget to call a corresponding log_exit.

log_exit/1

log_exit(_Tag) -> ok

Logs a message and signals exiting a function, block, ...

This function will first decrease the indentation level of logs and then print the log message (using the decreased indent). It should be paired with a call to log_enter.

set_filter_mode/1

set_filter_mode(Mode) -> ok

  • Mode = include | exclude

Sets the filtering mode.

When set to 'include' only tags registered with filter_set/1 will be printed. When set to 'exclude' tags registered with fileter_set/1 will NOT be printed.
