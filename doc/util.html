dereference/1

dereference(Variable) -> VarAddr

  • Variable = tVarType()
  • VarAddr = tVarAddr()

Translates a variable's usage node into an address.

Memory cells (either global or local) should be accessed only with addresses generated by this function.
Internally, the address format consist of the variable's name and eventual indices (if the variable is an array).

dereference_decl/2

dereference_decl(Context::{Scope, Manager, Memory}, VarDeclaration) -> VarAddress

  • Scope = pid()
  • Manager = pid() | no_manager
  • Memory = pid()
  • VarDeclaration = tVarDecl()
  • VarAddress = tVarAddr()

Translates a variable's declaration into all of its possible addresses.

Memory cells (either global or local) should be created only with addresses generated by this function.
Example: the following variable declaration: int[] foo[1..3];
will yield the following list of addresses: [ [1,"foo"], [2,"foo"], [3,"foo"] ].

reply/2

reply(Token, Response) -> ok

  • Token = {pid(), reference()}
  • Response = any()

Answers a send_sync's message.

Answers a message sent previously with send_sync/2 or send_sync/3.
The first element of a received message's tuple is a token. This token must be used in the response in order to answer in the same "communication channel".

Sends a synchronous message with timeout.

A call to this function sends the given message to the given recipient and waits until either:
